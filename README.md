# 🎵 Blindtest en ligne

Ce projet en **COURS DE DÉVELOPPEMENT** est un **jeu de blindtest multijoueur en ligne**, permettant à plusieurs utilisateurs de s'affronter en temps réel sur des extraits musicaux, grâce à l'API de **Deezer**.

## 🚀 Stack technique

- **Frontend** : [Vue.js](https://vuejs.org/)  
- **Backend** : [Express.js](https://expressjs.com/) & [Socket.io](https://socket.io/)  
- **API Musique** : [Deezer API](https://developers.deezer.com/)

## ⚙️ Installation et lancement du projet

### 1. Cloner le dépôt

```bash
git clone <url-du-dépôt>
cd <nom-du-dossier>

cd back
npm install
npm run start

cd front
npm install
npm run dev
```
